GameMaker Studio

GameMaker Studio Logo

GameMaker Studio is a proprietary game creation system created by Mark Overmars in the Delphi programming language.

GameMaker accommodates the creation of cross-platform and multi-genre video games using drag and drop action sequences or a sandboxed scripting language known as Game Maker Language, which can be used to develop more advanced games that could not be created just by using the drag and drop features.

GameMaker was designed to allow novice computer programmers to be able to make computer games without much programming knowledge by use of these actions.
